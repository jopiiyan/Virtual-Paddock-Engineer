## What this project is (one paragraph)

A full-stack F1 analytics app: a React dashboard compares two drivers' race telemetry, and
a RAG chat assistant answers natural-language questions grounded in that race data. The app
runtime is fully local (Ollama: `llama3.2` generator + `nomic-embed-text` embeddings, no API
keys). On top of the app sits a measured retrieval-evaluation harness that freezes a naive
dense baseline and ablates hybrid search (BM25 + RRF), multi-query expansion, and
cross-encoder reranking against it. The honest headline: on this small, clean corpus the
naive dense baseline is hard to beat, and only hybrid clearly earns its place.

## Key code paths

- `backend/ingestion.py` — pulls FastF1 data, chunks per driver/stint, embeds, writes to Supabase.
- `backend/retrieval/pipeline.py` — the config-driven retrieve() (dense + BM25 legs, RRF, optional MQ/rerank).
- `backend/api.py` — FastAPI app; routes retrieval through `vectorstore.dense_search`, streams answers over SSE.
- `eval/run_eval.py` — runs a config end to end, writes results, appends to `eval/results/runs.jsonl`.
- `eval/ragas_eval.py` — RAGAS answer-quality metrics judged by Gemini.
- `backend/tests/` — metric / RRF / pipeline unit tests.
- `configs/*.yaml` — one file per ablation config (baseline, bm25_only, hybrid, hybrid_mq, hybrid_rerank*, full).

## The headline result (so you don't misstate it)

Naive dense baseline is the strongest single config on recall@5 / MRR / nDCG@10. Hybrid is
the one keeper (closes the exact-term gap, lifts deep recall, near-zero added cost). Reranking
and multi-query each rescue one query type while regressing others, at real latency cost, and
do not beat the baseline on this corpus. An independent Gemini judge (RAGAS) corroborates it.
Every number is reproducible; nothing is estimated.

## Reproduce (fast path)

```bash
cd backend && pip install -r requirements.txt && cd ..
python eval/validate_golden.py            # golden-set integrity
python -m pytest backend/tests            # metric / RRF / pipeline tests
python -m eval.run_eval --config configs/baseline.yaml --out eval/results/baseline.json --no-ragas
python scripts/plot_tradeoff.py           # regenerate docs/tradeoff.png
```

Full ablation and RAGAS commands are in [README.md](README.md) and [EVALUATION.md](EVALUATION.md).
